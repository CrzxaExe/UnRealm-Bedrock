# Modifiarity

Same functionality with Minecraft Enchantment, but it can be costumisable

_list_

## Axe

- Life Parser (3)

`While do critical damage, deals additional damage by 8/16/20% target max health. Has cooldown: 5/4/3s`

- Shock (2)

`While do critical damage, target will be stun and get Weakness and Blindness effect for 1/2s`

## Hoe

- Experincer (5)

`Harvesting crops gain xp by 3/6/9/12/15`

- Replanter (1)

`Automatic replant crop when harvesting crops`

## Mace

- Half Life (3)

`Heal user by 3/5/8 while hitting target while falling`

## Pickaxe

- Chain Break (3)

`Break same neighbor block by 3/4/5 times`

- Explosion (1)

`Hitting enemy create explosion with level 1.5`

- Extractor (3)

`Has a chance to drop original block by 20/30/35%`

## Sword

- Benefit (3)

`Gain Speed and Strength effect level 1/2 for 2/4/5s every time kill target`

- Fire Spreader (3)

`Spread fire for 3/3/4s to all enemy within 1.5/2 block radius from target`

- Slashing (3)

`Attack will sweep all entity around target and deals 2/3/5 damage, cooldown 0.5s`

- Poison (2)

`Hitting target will apply Poison level 2/3 for 3s`

- Vampire (3)

`Hitting target will heal user by 1/2/3 hp`
